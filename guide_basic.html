1. 基本表示・モード切替 1. Basic Display / Modes

本ツールは、ユーザーの好みに合わせた2つの入力モードを備えています。 This tool features two input modes to suit user preferences.
画面上部のタブ（Prompt / Spreadsheet）でいつでも切り替えることができます。 You can switch between them at any time using the tabs at the top of the screen (Prompt / Spreadsheet).

📑 2つの入力モード 📑 Two Input Modes

📝 プロンプトエディタ (Prompt) 📝 Prompt Editor (Prompt)
WebUIで見慣れた、カンマ区切りのテキスト形式で入力するモードです。 A mode for entering text in the comma-separated format familiar from the WebUI.
向いている作業: Suitable for:
  • WebUIから既存プロンプトを貼り付ける
  • Pasting existing prompts from the WebUI
  • 文章のように一気に書き込む
  • Writing prompts all at once like a sentence
  • 完成したプロンプトを最終確認する
  • Final checking of the completed prompt
📊 スプレッドシート (Spreadsheet) 📊 Spreadsheet (Spreadsheet)
Excelのような表形式でタグを管理するモードです。 A mode for managing tags in a table format like Excel.
向いている作業: Suitable for:
  • タグをカテゴリごとに整理する
  • Organizing tags by category
  • 「髪」「衣装」「背景」など役割ごとに管理する
  • Managing by roles such as "hair," "outfit," or "background"
  • ExcelやGoogleスプレッドシートと連携する
  • Collaborating with Excel or Google Sheets

💡 ビジュアルエディタへの反映フロー 💡 Flow to Reflect in Visual Editor

本ツールの最大の特徴である「ビジュアルエディタ」を使用するには、以下の手順でデータを反映させます。 To use the "Visual Editor," which is the most significant feature of this tool, follow these steps to reflect your data.

1. データの同期 (Convert): 1. Data Synchronization (Convert):
各エディタの下部にある Press the ⬇️ ビジュアルエディタに反映 ⬇️ Reflect in Visual Editor ボタンを押すと、現在の入力内容が解析され、画面中央にタグブロックとして展開されます。 button at the bottom of each editor to analyze the current input and expand it as tag blocks in the center of the screen.
2. 編集の書き戻し (Reflect): 2. Writing Back Edits (Reflect):
ビジュアルエディタで並び替えや削除を行った後、以下の青いボタンを押すことで、現在の入力モードに編集結果が書き戻されます。 After rearranging or deleting tags in the Visual Editor, press the following blue buttons to write the edit results back to the current input mode.

プロンプトエディタ使用時: When using Prompt Editor: ⬆️ プロンプトエディタに反映 ⬆️ Reflect in Prompt Editor

スプレッドシート使用時: When using Spreadsheet: ⬆️ スプレッドシートに反映 ⬆️ Reflect in Spreadsheet

🌐 言語設定と表示モード 🌐 Language Settings and Display Modes

画面右上のヘッダーコントロールを使用して、表示をカスタマイズできます。 You can customize the display using the header controls at the top right of the screen.

言語の切り替え (Language) Language Switching (Language)

🌐 EN/JP ボタンをクリックすると、メニューやボタンの表示言語（日本語 / 英語）が切り替わります。 Clicking this button switches the display language of menus and buttons (Japanese / English).
※プロンプトの内容自体（タグの翻訳など）は変化しません。 *The content of the prompt itself (such as tag translations) does not change.

ダークモード (Dark Mode) Dark Mode (Dark Mode)


チェックボックスで、画面全体の背景色を切り替えられます。 You can switch the background color of the entire screen with the checkbox.
長時間の編集作業を行う際は、ON（ダークモード）にすることで目の負担を軽減できます。 When working on edits for a long time, turning it ON (Dark Mode) can reduce eye strain.

📌 ヒント: 📌 Hint:
これらの設定（言語、ダークモード）および入力中の内容は、ブラウザに自動保存されます。 These settings (language, dark mode) and the content being entered are automatically saved in the browser.

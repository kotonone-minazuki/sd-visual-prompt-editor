3. タグ検索・追加 3. Tag Search / Add

画面右側（スマホの場合は下部）にある「タグ検索パレット」を使うと、Danbooruデータセットに含まれる膨大なタグの中から、必要なものを探してプロンプトに追加できます。 Using the "Tag Search Palette" on the right side of the screen (or bottom on smartphones), you can search for and add tags to your prompt from the vast collection in the Danbooru dataset.

🔍 タグの検索 (Search) 🔍 Tag Search (Search)

検索ボックスにキーワードを入力すると、リアルタイムで候補が絞り込まれます。
以下の3種類のデータに対して検索が行われます。 Entering keywords in the search box filters candidates in real-time.
Searches are performed across the following three types of data:

  • タグ名 (Tag): Tag Name (Tag): 英語のタグ名 (例: 1girl) English tag name (e.g., 1girl)
  • 日本語訳 (JP): Japanese Translation (JP): 日本語の意味 (例: 女の子) Japanese meaning (e.g., 女の子)
  • 翻訳エイリアス (Trans): Translation Alias (Trans): その他の翻訳候補 Other translation candidates
💡 ポイント: 💡 Point:
「女の子」のように日本語で入力してもヒットするため、英語のタグ名がわからなくても直感的に探すことができます。 Since it hits even when entered in Japanese (e.g., "女の子"), you can search intuitively even if you don't know the English tag name.

🖱️ タグの追加 (Drag & Drop) 🖱️ Adding Tags (Drag & Drop)

検索結果に表示されたタグは、ビジュアルエディタへ直接ドラッグ＆ドロップして追加します。 Tags displayed in the search results can be added by dragging and dropping them directly into the Visual Editor.

検索結果の見方 Reading Search Results

  • 上段: Top: 正式なタグ名（プロンプトに使われる文字） Official tag name (characters used in the prompt)
  • 下段: Bottom: 日本語訳などの補足情報 Supplementary information such as Japanese translation
  • 右側: Right: 出現件数（人気度により色が変わります） Count (color changes based on popularity)
追加手順 Addition Steps
  1. 欲しいタグをマウスで掴みます（スマホは長押し）。
  2. Grab the desired tag with your mouse (long-press on smartphones).
  3. 左側のビジュアルエディタ上の「入れたい場所」まで引っ張ります。
  4. Drag it to the "desired position" in the Visual Editor on the left.
  5. ドロップすると、その位置にタグが挿入されます。
  6. Drop it to insert the tag at that position.
📋 クリックでコピー: 📋 Click to Copy:
タグをクリックすると、クリップボードにタグ名がコピーされます。他のツールに貼り付けたい場合に便利です。 Clicking a tag copies the tag name to the clipboard. Convenient for pasting into other tools.

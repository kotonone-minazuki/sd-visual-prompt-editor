    デザインシステム (Design)

    本プロジェクトの視覚的定義と、CSS 変数によるテーマ管理について解説します。

    1. カラーパレット (Theme Colors)

    本システムは CSS 変数を利用しており、ライトモードとダークモードの切り替えに柔軟に対応しています。

    基本システムカラー

    変数名 用途 デフォルト値 (Dark)
    --bg-color 全体の背景色 #1a1a1a 前後
    --text-color 標準テキスト #ffffff
    --accent-color 強調表示・ボタン プロジェクト固有のアクセント色
    --box-bg 入力エリア等の背景 モードに応じたボックス色

    2. タグの視覚的定義 (Tag Styles)

    Stable Diffusion のプロンプトを構成する要素を、枠線や文字色で直感的に識別できるようにしています。

    カテゴリ別スタイル

    タグ種別 枠線・文字色 意味・用途
    LoRA / 制御タグ #e84393 (ピンク) BREAK, AND, LoRA等の特殊制御
    コメントタグ #27ae60 (緑) # で始まるメモ・コメント
    複合タグ #6c757d (グレー) 複数の要素を組み合わせたタグ
    標準タグ --tag-bg (#2b2b2b) 一般的な Danbooru タグ

    3. UI コンポーネントの挙動

    • ホバー効果: 提案エリアのタグにマウスを合わせると、アクセント色や不透明度が変化し、選択可能であることを示します。
    • ダークモード対応: body クラスの切り替えにより、すべての CSS 変数が一括で更新される設計になっています。

    4. 今後の拡張予定

    • タグの重み可視化: プロンプト内の (tag:1.2) といった重みを、色の濃淡やサイズで表現する機能の検討。
    • カスタムテーマ: ユーザーが styles.css を編集することなく、お気に入りのカラーテーマを選択できる仕組み。

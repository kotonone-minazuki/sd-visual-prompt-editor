# **SD Visual Prompt Editor 開発者ドキュメント**

本プロジェクトの内部構造、デザインシステム、および開発環境のセットアップに関するガイドです。

## **🚀 開発環境のセットアップ (Getting Started)**

本アプリケーションは静的な HTML/JavaScript で構成されているため、index.html をブラウザで直接開くことでも動作しますが、ローカルサーバーを介した実行を推奨します。

### **1\. ローカルサーバーの起動 (start_server.bat)**

Windows 環境での開発・動作確認を容易にするため、ルートディレクトリにバッチファイルを用意しています。

- **ファイル**: start_server.bat
- **仕組み**: Python の標準モジュールである http.server を利用して、ポート 8000 で簡易 HTTP サーバーを立ち上げます。
- **実行手順**:
  1. Python がインストールされていることを確認します。
  2. start_server.bat をダブルクリックして実行します。
  3. ブラウザで http://localhost:8000 にアクセスします。

### **2\. 言語設定 (Language Settings)**

本アプリは日本語（JP）と英語（EN）の多言語表示に対応しています。

- **切り替え方法**: UI 右上の「🌐 EN/JP」ボタンから即座に切り替えが可能です。
- **保存の仕様**: ユーザーが選択した言語設定はブラウザの localStorage (キー名: lang) に保存されます。
- **初期挙動**: 初回訪問時は localStorage の値を確認し、存在しない場合はデフォルトとして「日本語 (ja)」が適用されます。
